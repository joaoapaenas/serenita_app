# App Walkthrough

This section provides a high-level overview of the application's screens and core user workflows.

## Main Screens

1.  **Welcome Screen:** The initial screen where users can create or select a profile.
2.  **Cycle Manager:** The main dashboard for creating, editing, and managing study cycles.
3.  **Study Session:** The focused interface for completing a study session, including tasks and a timer.
4.  **Performance Dashboard:** Visualizations and analytics to track study progress over time.

## Key Workflows

*   **Onboarding:** Creating a new user profile.
*   **Cycle Creation:** Defining a new study cycle with subjects and goals.
*   **Studying:** Starting and completing a session from the study queue.
*   **Reviewing Performance:** Checking analytics to see how you're doing.
